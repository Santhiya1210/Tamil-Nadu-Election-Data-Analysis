# Tamil Nadu Election Data Analysis
# Project Overview
This project analyses Tamil Nadu election data using Power BI.
The dashboard shows candidate details, vote distribution, turnout percentage, and winning margins in a simple and clear way.
# Data Source
Tamil Nadu Election Dataset
Source: Open data portal
Year: 2025
# Problem Statement
To analyse party-wise and constituency-wise election results
To understand vote distribution and winning margins
To identify close fights and strong wins
To present insights using Power BI dashboards
# Tools Used
Power BI
# Data Processing
Removed null values and corrected data types
Created calculated columns like Age Group and Margin Category
Created measures for total candidates and total votes
Rounded values (K, M) are used only for display purpose
# Dashboard Insights
Chennai has the highest number of candidates and total votes
Tiruvallur shows higher voter turnout percentage
Male candidates received more votes than other categories
Older and middle age groups show higher participation
Most constituencies fall under close margin category
# Conclusion
This dashboard gives a clear understanding of Tamil Nadu election results.It helps analyse voter behaviour, participation levels, and competition across districts and constituencies, supporting better decision-making for future elections.
